Editing Tribes

Tribe names are mapped from what Sage outputs to a different name in the Tribal supporting document. These mappings can be changed by editing TribeList.xlsx in the SageToPDF\Templates\ folder (typically found at C:\Program Files (x86)\Sage To PDF).

Only two columns are used by the program. Values in all other columns are ignored.

  • Sage Tribal List

    The name that Sage will export to Excel. This should exactly match everything before the "Item:" in the "Trans Descriptions" column.

    Sage truncates anything beyond 30 characters, this program mimics that by ignoring anything in this column beyond 30 characters.

    E.G.: "Lac du Flambeau Band of Lake Superior" is shortened to "Lac du Flambeau Band of Lake S" while searching.

  • Tribal

  • END

    This signifies the end of the list of tribes to load.

    WARNING: Without this the program will keep reading the spreadsheet until it crashes.
